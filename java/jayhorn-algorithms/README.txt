BellmanFord
-----------
Meanings of the trailing terms / numbers:
  - Fun   -> tests functional properties
  - Mem   -> tests memory safety
  - Sat   -> program is correct
  - Unsat -> program is incorrect
  - 01    -> uses 1-D flattened matrices
  - 02    -> uses 2-D matrices
